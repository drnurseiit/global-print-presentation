О проекте

Что это и для кого

Global Print CRM — это система для управления полиграфическим производством «под ключ». Разработана для типографий полного цикла: от приёма заказа до отгрузки готовой продукции.

📋Приём и ведение заказов
🎨Дизайн и согласование
🔩Пластины и печать
🏭Цех и постпечать
📦Склад и остатки
💰Финансы и отчёты

Ключевая идея

Один интерфейс для всех отделов: менеджер создаёт заказ → дизайнер правит макет → верстальщик готовит спуск → печатник печатает → цех обрабатывает → склад отгружает. Каждый этап — со своими правами и задачами.

Технологии

Стек технологий

Современный изолированный стек для надёжной работы в production.

⚛️

Frontend

React 18 + TypeScript + Vite. React Router, Storybook, Vitest. Адаптивная вёрстка, CSS Variables для темизации.

🖥️

Backend

NestJS + TypeORM. Модульная архитектура (auth, orders, clients, plates, files...). JWT + Passport.

🗄️

База данных

PostgreSQL 16. TypeORM миграции, связи между сущностями, кастомные запросы.

🐳

Инфраструктура

Docker Compose. 3 сервиса: nginx (frontend) → nest (backend) → postgres. Готово к деплою.

🔐

Безопасность

Ролевая модель (9 ролей). 60+ разрешений. JWT токены, bcrypt пароли, доступ по отделам.

📊

UI/UX

Тёмная/светлая тема, канбан-доска, анимации, кастомный скроллбар, скелетоны, тултипы.

Архитектура

Как это работает

01

Заказ

Менеджер создаёт заказ, заполняет параметры изделия, сроки, финансы. Прикрепляет файлы.

02

Дизайн → Верстка

Дизайнер готовит макет, загружает в заказ. Верстальщик делает спуск полос.

03

Пластина → Печать

Объединение заказов в пластины. Печатник запускает тираж, контролирует качество.

04

Цех → Готово

Резка, ламинация, упаковка. Заказ готов к выдаче. Архив для аналитики.

«Каждый отдел видит только свои задачи, но вся картина производства доступна в реальном времени на доске заказов.»
